# c-clothingmall
## 一、简介：

基于 Vue 2.x 编写的女性服装商场移动端网页。

## 二、上手指南：

#### 1.运行项目


```bash
# cd c-clothingmall
# npm install 
# npm run serve
```

此时在浏览器输入：localhost:8080 即可访问到项目。

#### 2.项目打包

```bash
# npm run build
```
